# AI 기반 엑셀-PPT 자동 변환 도구 (Proprea)

엑셀 데이터를 지능적으로(Semantic Chunking) 논리 단위로 분할하여, 사용자가 지정한 PPT 템플릿의 표 서식과 크기를 유지한 채 자동으로 데이터를 채워주는 Windows GUI 데스크톱 파이썬 애플리케이션입니다.

## 주요 기능

- **직관적인 GUI**: `tkinter` 기반의 데스크톱 UI를 통해 엑셀 데이터소스와 PPT 템플릿, 그리고 결과물 저장 위치를 윈도우 탐색기 메뉴로 손쉽게 지정할 수 있습니다.
- **AI 의미론적 분할 (Semantic Chunking)**: 구글 최신 모델인 Gemini 2.5 Flash (`google-genai` 패키지)를 활용해, 엑셀 데이터를 단순히 줄 수로만 자르지 않고 문맥(시험 평가 항목, 테스트 Phase, 대상 Component 등)에 맞춰 끊기지 않게 분할합니다.
- **템플릿 정밀 타겟팅**: `1-1` (1번 슬라이드의 첫 번째 표) 처럼 슬라이드 번호와 표 번호를 결합하여 데이터를 주입할 위치를 자유롭게 제어할 수 있습니다.
- **엑셀 데이터 범위 지정 기능**: `A1:F20`과 같은 표준 문자열 포맷으로 읽어들일 데이터 범위를 조절할 수 있습니다 (입력하지 않으면 전체). 
- **100% 원본 템플릿 서식 보존 방어 로직**: 데이터가 들어가면서 표의 비율이나 높이, 셀 색상이 틀어지는 것을 막기 위해 `TextFrame`과 내부 단락(Paragraph) 뼈대를 건드리지 않고 텍스트만 주입(`add_run()`)하는 고급 설계가 적용되었습니다. (텍스트의 가독성을 위해 결과물은 검정색의 9pt 사이즈로 고정 치환 적용됨)
- **오동작 방지 및 안전장치 (Fail-Safe)**: 
  - 권한 문제(`Permission Denied`) 방지: 엑셀 파일을 다른 곳에서 열어두고 구동 시도 시, 즉시 멈추고 안내 메시지를 띄웁니다.
  - PPTX 파일 파손 및 누락 감지(`Package not found`): 잘못된 파일을 선택했을 때 에러 핸들링을 통해 올바른 파일을 고르도록 유도.
  - Abort 기능: 작업 무한루프 및 오류 시 OS 단에서 즉시 프로세스를 소멸시키는 붉은색 강제 종료 장치를 배치했습니다.

## 설치 및 실행 가이드

1. Python 환경에서 필요한 모듈을 설치합니다.
```bash
pip install pandas python-pptx openpyxl google-genai
```

2. 프로젝트 폴더로 이동하여 파이썬 스크립트를 실행합니다.
```bash
python excel_ppt_converter.py
```

*참고: 원활한 구동을 위해서는 파이썬 코드 리스트 상단의 Gemini API 키가 유효해야 합니다.*
